# Модель-Представление-Контроллер

Модель-Представление-Контроллер (или для краткости: MVC) - это популярный паттерн проетирования (типовое решение), используя который разделяется источник данных (Модель) от шаблонов представления (Представление) и логики запросов (Контроллер).

Это делает процесс разработки легче и максимализирует повторное использование кода. То есть, позволяет писать намного меньше кода.
